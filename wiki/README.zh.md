# Beadline（珠链）— 介绍

Beadline 是一款音乐播放器与曲库管理工具。与普通文件播放器不同，它以 **Song Unit（曲目单元）** 为核心来组织音乐——一个可以容纳多个媒体源、丰富标签和播放偏好的逻辑实体。

---

## Song Unit（曲目单元）

Song Unit 是你在 Beadline 中管理的核心对象。它代表一首歌，但不绑定到单个文件。

一个 Song Unit 可以包含：
- 多个画面源（MV、现场版、静态封面……）
- 多个音频轨道（原唱、纯音乐、不同录音版本……）
- 伴奏音轨（卡拉 OK 伴奏）
- 歌词文件（LRC 格式，带时间轴同步）
- 元数据与标签

任意时刻，每种输出类型只有一个源在播放。你可以在播放过程中切换源，不会丢失当前进度。

这意味着你可以把同一首歌的 MV、现场版和卡拉 OK 版都放在同一个 Song Unit 里，随时切换。

---

## 源（Source）

源是附加到 Song Unit 的一段媒体。共有四种类型：

| 类型 | 说明 | 输出 |
|------|------|------|
| 画面 | 视频或图片内容 | 显示 |
| 音频 | 主音轨 | 音频 |
| 伴奏 | 卡拉 OK 或纯音乐音轨 | 音频（与音频互斥） |
| 歌词 | LRC 同步歌词文件 | 歌词叠加层 |

源可以是本地文件、直链 URL，或通过配置的 API 桥接获取的在线平台内容。

当一个 Song Unit 有多个同类源时，你可以设置优先级顺序。Beadline 默认使用最高优先级的源，也可以在播放时手动切换。

---

## 输出（Output）

Beadline 有三个输出通道，每个通道同一时刻只有一个源激活：

- **显示** — 屏幕上显示的内容（画面源，可以是视频或图片）
- **音频** — 你听到的声音（音频源或伴奏源，二选一）
- **歌词** — 同步歌词，显示在屏幕上或悬浮窗口中

---

## 标签（Tag）

标签是你整理和查找 Song Unit 的方式。共有三类：

### 内置标签

始终存在的标准元数据字段：

- `name` — 歌曲标题
- `artist` — 演唱者
- `album` — 专辑名称
- `time` — 年份或年代
- `duration` — 时长（秒）

### 用户标签

你自己创建的标签。支持：

- **层级关系** — 标签可以有父子关系。搜索父标签时，默认包含所有子标签。
- **别名** — 一个标签可以有多个名称（如跨语言）。任意别名都可以在搜索中使用，解析到同一个标签。

层级示例：
```
luotianyi
├── luotianyi/v4
│   └── luotianyi/v4/meng
└── luotianyi/live
```

搜索 `tag:luotianyi` 会匹配以上所有标签。

### 自动标签

由系统维护，不可手动编辑：

- `user:xx` — 谁添加或点播了这首歌
- `playlist:xx` — 这首歌属于哪个播放列表

---

## 播放列表、分组和队列都是标签

这是 Beadline 的核心设计之一：**播放列表就是标签**。

当你把一首歌加入名为「收藏」的播放列表时，Beadline 会给它打上 `playlist:收藏` 标签。没有独立的播放列表对象——一切都在同一套标签系统里。

播放队列也是如此。队列中的歌曲可以携带 `user:xx` 标签，记录谁点播了它，这也是点歌系统的基础。

这意味着播放列表、分组、点歌记录——全部都可以通过同一个标签界面来搜索、筛选和管理。

---

## 搜索

Beadline 有一套查询表达式语言用于搜索曲库。直接在搜索栏中输入表达式即可。

默认行为：裸关键词按名称搜索。
```
hello  →  name:*hello*
```

一些示例：
```
artist:luotianyi              # 洛天依的歌
-album:singles                # 排除「singles」专辑
time:[2017-2024]              # 2017 到 2024 年的歌
artist:luotianyi OR artist:yanhe   # 任意一位歌手
(tag:live OR tag:mv) artist:luotianyi  # 分组条件
```

标签别名是透明的——用任意别名搜索都能找到相同的结果。

---

## 歌词

歌词是一种源类型（LRC 格式）。有三种显示模式：

- **关闭** — 不显示歌词
- **屏幕** — 显示在主显示区域，支持 KTV 风格（当前行 + 下一行）或滚动风格
- **悬浮** — 系统级置顶悬浮窗口，可拖动和调整大小，覆盖在其他应用之上

---

## 在线源

Beadline 以本地文件为主，但支持在线源作为补充：

- 直链 URL（MP3、MP4、HLS、DASH）
- 通过自托管 API 桥接的平台内容（哔哩哔哩、网易云音乐等）

Beadline 不直接调用第三方 API。你需要配置自己的桥接服务器并填写地址——由你掌控隐私和合规性。

---

## 导入与导出

Song Unit 可以导出为 ZIP 文件，包含元数据和本地源文件。批量导出将多个 Song Unit 打包为一个压缩包。文件按内容哈希去重，共享文件不会重复存储。
