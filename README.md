# Beadline (珠链)

> ⚠️ **Vibe Coding Project / 氛围编程项目**
>
> This project is developed with AI-assisted "vibe coding". The codebase may contain incomplete features, experimental designs, or rough edges. Use with caution and do not rely on it for production or critical use cases.
>
> 本项目为 AI 辅助的「氛围编程」项目，代码库可能包含未完成的功能、实验性设计或不稳定之处。请谨慎使用，不建议用于生产环境或关键场景。

A cross-platform music player and library manager built around the **Song Unit** abstraction and a powerful tag-based query system.

跨平台音乐播放器与曲库管理工具，以 **Song Unit（曲目单元）** 抽象模型和强大的标签查询系统为核心。
